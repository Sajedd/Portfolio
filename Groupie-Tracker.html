Pésentation du projet Groupie Tracker - API Go Fyne

Le Groupie Tracker est une application API Go Fyne qui permet de suivre les groupes de musique préférés et leurs tournées. L'application utilise une interface utilisateur simple et intuitive pour afficher les informations des groupes, des salles de concert et des dates de tournée.

Le code source du Groupie Tracker est disponible sur Github à l'adresse suivante :“https://github.com/Sajedd/Groupie-Tracker.git”
